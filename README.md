# Chrome Dino Game 
This is my version of [Chrome Dino Game](https://chromedino.com/)

## Technologies
Project is created with:
* Python 3.9.6
* Pygame 2.1.2

## How to run?
```
$ https://github.com/mazurmilosz000/Chrome_Dino.git
$ pip install -r requirements
```

## Controls
| Key | Usage |
| :---:   | :-: |
| ESC | Pause Menu |
| UP| Dino Jump |
| DOWN | Dino Duck |
